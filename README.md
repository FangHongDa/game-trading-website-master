# 基于springboot在线游戏商城系统

#### 介绍
基于springboot在线游戏商城系统，前端采用layui+js，主要分两端Web端与管理后端，代码规范整洁，技术选型也非常经典，合适初学java编程者学习以及计算机专业学生。


#### 软件架构
前端：layui+js | jquery | bootstrap  
后端：spring | springmvc | mybatis  
环境：jdk1.8 | maven | mysql | tomcat
